# Generative-AI-Based-Mock-Interviewer
A Generative AI-Based Mock Interviewer is an advanced interview preparation platform designed to help students, graduates and professionals improve their interview skills through artificial intelligence. The system simulates real world interview experiences by conducting interactive sessions and evaluating candidate responses in real time.

<img width="1536" height="1024" alt="AI Mock Interviewer" src="https://github.com/user-attachments/assets/5a02dc2a-defa-4159-892a-5105391e5531" />

---

**AI Mock Interviewer** is an intelligent interview preparation platform powered by **Flask**, **Python** and the **OpenAI API**. It simulates real interview sessions, evaluates user responses, provides instant AI feedback, tracks performance and helps users improve their communication and technical interview skills.

# ⚙️ Installation

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/AI-Mock-Interviewer.git
```

## 2. Navigate to the Project Folder

```bash
cd AI-Mock-Interviewer
```

## 3. Create a Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

## 4. Install Dependencies

```bash
pip install -r requirements.txt
```

## 5. Configure Environment Variables

Create a `.env` file:

```env
OPENAI_API_KEY=your_openai_api_key
FLASK_APP=app.py
FLASK_ENV=development
```

## 6. Run the Application

```bash
flask run
```

or

```bash
python app.py
```

The application will be available at:

```text
http://127.0.0.1:5000
```
